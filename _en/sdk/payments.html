Api payment methods using sdk  
PayU provides an API that allows your shop to process different types of transactions with multiple payment means depending on the selected country.  
If you want to integrate with our API you must point the requests to the following URLs:  
  • SDK Java
  • SDK PHP
PayU.paymentsUrl = “https://api.payulatam.com/payments-api/”;
PayU.reportsUrl = “https://api.payulatam.com/reports-api/”;
Environment::setPaymentsCustomUrl(“https://api.payulatam.com/payments-api/4.0/service.cgi”);
Environment::setReportsCustomUrl(“https://api.payulatam.com/reports-api/4.0/service.cgi”);
Environment::setSubscriptionsCustomUrl(“https://api.payulatam.com/payments-api/rest/v4.9/”);
 
The available methods for the payment API are: PING, ACTIVE PAYMENT METHODS QUERY, and TRANSACTION PROCESSING. Below is a description of each.
 
 
 
Transaction processing
 
We have different methods for processing transactions:

All available operations are:

  • Authorization: used to verify if a credit card is active, if it has money, etc. The transaction is not complete until a transaction capture is sent (only available for accounts Brazil).
  • Capture: terminates a previously authorized transaction. This is when the debit to the card’s account is performed (only available for accounts in Brazil).
  • Authorization and capture: : this is the most used transaction type. This option sends the transaction amount to authorization and it is approved immediately after capturing.
  • Cancellation: Used to reverse a previously authorized transaction (only available for accounts in Brazil).
  • Refund: Used to reverse a previously captured transaction.

Query of active payment means
 
You can use the getPaymentMethods method to obtain a list of active payment means on our platform.
  • SDK Java
  • SDK PHP
Request example: 
List<PaymentMethodComplete> response = PayUPayments.getPaymentMethods();
LoggerUtil.info("{0}", response);;
Request example: 
$array=PayUPayments::getPaymentMethods();
$payment_methods=$array->paymentMethods;
foreach ($payment_methods as $payment_method){
	$payment_method->country;
	$payment_method->description;
	$payment_method->id;
}
