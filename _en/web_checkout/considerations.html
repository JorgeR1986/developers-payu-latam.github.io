Required variables
  1. CUS: It is a variable that is sent in the response page; unique tracking code is the payment reference within the Bank, it applies only to payments with PSE.
  2. pseBank: It is a variable that is sent in the response page; it is the name of the bank, it applies only to payments with PSE.
  3. pseCycle: It is a variable that is sent in the response page, it is the identifier generated by PSE.
  4. pseReference1 / pseReference2 / pseReference3:: Are variables that sent in the response page, they are the reference numbers for payments with PSE.

Required variables
