Variáveis enviadas para o gateway

Campo Tipo Tamanho Descrição Requerido em
amount Numérico 14 É o montante total da transação. Pode conter dois dígitos decimais. Por exemplo: 10000.00 ou 10000. Todos os países
merchantId Numérico 12 É o número identificador de seu comércio no sistema do PayU, este número é encontrado no e-mail de criação da conta. Todos os países
referenceCode Alfanumérico 255 É a referência da venda ou pedido. Deve ser único por cada transação enviada ao sistema. Todos os países
accountId Numérico 6 Identificador da conta do usuário para cada país associado ao comércio, ao enviá-la somente abre-se as formas de pagamento pertencentes ao referido país. Todos os países
description Alfanumérico 255 É a descrição da venda. Todos os países
tax Numérico 14,2 É o valor do IMPOSTO da transação, se for enviado o IMPOSTO nulo, o sistema aplicará automaticamente 19%. Pode conter dois dígitos decimais. Por exemplo: 19000.00. Caso não incida IMPOSTO deve ser enviado como 0. Todos os países
taxReturnBase Numérico 14,2 É o valor base sobre o qual se calcula o IMPOSTO. Caso não incida IMPOSTO, deve ser enviado como 0 Todos os países
signature Alfanumérico 32 É a assinatura digital criada para cada uma das transações. Todos os países
