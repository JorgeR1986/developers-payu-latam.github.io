Integrações PayU.

Escolha a opção mais adequada para suas necessidades:

Integração API

Projete seu próprio checkout e gerencie seu sistema de pagamentos.

Acesse serviços adicionais, como tokenização ou pagamentos recorrentes.

O cliente permanecerá em seu web site durante a compra, sua marca será a protagonista.

Integração SDK

Acesse serviços adicionais, como tokenização ou pagamentos recorrentes.

Disponível para os linguagens JAVA y PHP.

Projete e personalize seu própio checkout, o cliente permanecerá em seu web site durante a compra.

Saiba mais

Integração WebCheckout

PA PayU gerencia o sistema de pagamentos.

Integra o serviço de maneira mais simples, reduz os tempos de implementação.

A certificação PCI DSS é mais fácil de obter, a PayU captura os dados dos cartões de crédito.
