Puede contener dos dígitos decimales, si el valor no es enviado y no se tiene definido en la configuración por defecto de la cuenta se define el valor de la orden. Si se desea que este valor sea 0 se debe enviar 0 para las integraciones personalizadas o se debe definir el valor 0 en la configuración por defecto de la cuenta.
shippingValue Numérico 14 Valor del envío.

Si el valor de este campo es "0" (cero), se interpreta que el comercio asume todo el valor del envío calculado.

Si no viene ningún valor en este campo, el valor del envío será calculado y se aplicará la configuración de la cuenta para distribuir el valor calculado entre comercio y pagador.

Si el valor de este campo es mayor a cero, se interpreta que el comercio asume este valor, adicionalmente si el valor calculado supera este valor, se aplicará la configuración definida para la cuenta para distribuir el valor de la diferencia entre el comercio y el pagador y si el valor calculado es menor al valor enviado se asume el valor enviado.
